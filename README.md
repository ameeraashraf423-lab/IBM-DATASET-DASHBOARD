📊 IBM Sales Report Dashboard

This project presents an interactive Sales & HR Analytics Dashboard built to analyze employee data, income distribution, and workforce trends within an organization. The dashboard provides key insights into employee demographics, job satisfaction, attrition, and departmental performance.

🚀 Overview

The dashboard visualizes critical business metrics such as:

Total Monthly Income: 9M
Total Employees: 1383
Attrition Count: 226
Sum of Job Levels: 2774

It enables users to explore patterns and relationships across different departments, roles, and employee attributes.
